# 💸 neon flow pro — Personal Finance Tracker Chrome Extension

A lightweight Chrome extension designed to help users track spending, monitor financial habits, and stay in control of their personal finances directly from the browser.

Built for simplicity, speed, and accessibility, FinTrack enables users to log expenses, categorise transactions, and visualise spending behaviour without leaving their workflow.

---

## 🚀 Live Features

### 📊 Expense Tracking
Quickly log daily expenses with:

- Amount
- Category
- Description
- Date & time tracking

---

### 🗂 Smart Categorisation
Organise transactions into categories such as:

- Food
- Transport
- Shopping
- Bills
- Entertainment
- Savings
- Custom categories

---

### 📈 Spending Insights
Monitor financial behaviour through:

- Total expenditure overview
- Category-based breakdowns
- Spending trends

---

### 💾 Persistent Local Storage
All user data is securely stored using Chrome local storage.

No external database required.

---

### ⚡ Browser-Native Access
Accessible instantly from the Chrome toolbar for fast expense logging.

## 📦 Installation Guide

### Option 1: Install from Source (Developer Mode)

### Step 1: Clone the repository

```bash
git clone https://github.com/yourusername/fintrack-extension.git
```

---

### Step 2: Open Chrome Extensions

Navigate to:

```plaintext
chrome://extensions/
```

---

### Step 3: Enable Developer Mode

Toggle **Developer mode** in the top-right corner.

---

### Step 4: Load the extension

Click:

**Load unpacked**

Then select the project folder.

---

### Step 5: Pin the extension

Click the Extensions icon in Chrome and pin **FinTrack** for easy access.

---

## 📖 How to Use

### Adding an Expense

1. Click the FinTrack extension icon
2. Enter:
   - Expense amount
   - Category
   - Optional description
3. Click **Add Transaction**

The expense is instantly saved.

---

### Viewing Spending Data

Open the extension popup to:

- View all logged expenses
- Track totals
- Analyse category spending

---

### Managing Transactions

Users can:

- Delete entries
- Edit existing expenses
- Reset all stored data (if enabled)
